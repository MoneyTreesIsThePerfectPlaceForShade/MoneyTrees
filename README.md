## Дневник разработки
### `15 марта 2026`
Заметил дефектное поведение: при переходе на другую страницу сбрасывается тема фона на стандартную. Сама тема остается прежней, но цвет фона сбрасывается.

Контекст: в приложении комплексный градиентный фон с плавным переключением темы. Для реализации плавного переключения пришлось создавать `@property` в `CSS` модуле, а в `React` компоненте задавать цвет этой переменной. И чтобы переиспользовать `CSS` переменные из `shared/styles/colors.css`, я вытаскивал их через `ref` и `getPropertyValue` у элемента и в `useEffect` менял цвета в соответствии с темой.

Проблема: при переходе на другую страницу эти `CSS` переменные были `undefined` в тот момент, когда выполнялся `useEffect`, и `useLayoutEffect` тут не помощник.

Решение: сначала я пробовал задать разные `CSS` классы для тем, чтобы не приходилось использовать `@property` и задавать цвета в `JS` - сломалось плавное переключение темы. Отказывать от плавного переключения я не хотел, решил отказаться от получения значения `CSS` переменных из `React Ref`, т.е. отказавшись от переиспользования `CSS` переменных из `colors.css`. Неприятно, но не страшно, код будет чище, главное не забывать, что цвета фона живут не в `color.css`, где все остальные переменные для цвета, а в компоненте `PageLayout`. Решение, конечно, компромисное, но лучшего я не нашел на текущий момент.
